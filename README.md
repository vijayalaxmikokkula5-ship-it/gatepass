# 🚪 Gate Pass Management System
## 📌 Project Overview
The Gate Pass Management System is a web-based application developed as a **college group project** to simplify the process of requesting, approving, and managing student gate passes. The system helps reduce paperwork by providing a digital platform for students, administrators, and gatekeepers.
## ✨ Features
- Student Gate Pass Request
- Admin Approval and Rejection
- Gatekeeper Verification
- User-Friendly Interface
- Easy Navigation
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Git
- GitHub

## 🚀 How to Run
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the server using `node main.js`.
4. Open the frontend in your browser.
